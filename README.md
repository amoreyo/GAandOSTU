# OSTU

大津法（OTSU）是一种确定图像二值化分割阈值的算法，由日本学者大津于1979年提出。

## 流程

1. 用OTSU实现单阈值的分割（不用遗传算法）
2. 实现多阈值的分割（使用遗传算法）
   1. 设置分割为n个部分
   2. 确定直方图
   3. 初始化多种阈值
   4. 计算每个阈值之间的数学期望，类内方差；以及每个类之间的类间方差
   5. 求算单个类与其他类之间的类间方差的和，除上类内方差得到比较值
   6. 通过比较值筛选出优的阈值
   7. 变异随机生成新的阈值，优的阈值之间进行杂交（排序，对应取平均）
   8. 重复epoch遍step（4-7）
